Arak Digital Dental Clinics

The Project is developed for a Dental Clinic. The website contains clinic information, and it helps with scheduling and managing appointments for patients. Moreover, dentists can manage their patients and the clinic. The website built in Arabic Language.

Technologies :

It is a full stack application using Ruby on Rails and Mongoid.

  • Ruby on Rails: The framework Was used to build the website.
  • Ruby: The programming language used for the project.
  • Mongoid: Was used for database, It's a framework for MongoDB in Ruby.
  • Javascript: Was used for the functionality, using DOM manipulation.
  • JQuery: Was used for the functionality, using DOM manipulation.
  • Devise:Was used for authentication.
  • Source Control: Was used to manage commit versoins.
  • Visual Studio: Was used for coding.
  • Google Chrome Web Browser: Was used to view the website.
  • Bootstrap: The library used for styling.
  • Google Font: Was used for the website font.
  • Flat Pickr: The library used for Calendar in Appointment.
  • Png Tree Was used for emojis.
  • Adobe XD Was used for Wireframe.
Models :
  • Appointment.
  • Doctor.
  • User.
Relationship :
  • Appointment belongs to Doctor.
  • Appointment belongs to User.
  • Doctor has many Appointments.
  • Doctor has and belongs to many Users.
  • User has many Appointments.
  • User has and belongs to many Doctors.
Features :

The website built in the Arabic language as the client request. The Admin authorized to Creat, Read, Update and Delete (CRUD) Doctors and Appointments. Customers authorized to Creat, Read, Update and Delete (CRUD) their appointments and profile. The website has authentication for names, Signup, Sign in, sign out, and appointments.

  • There are two types of users, Admin and Regular User.
  • Users authorized to Signup, Sign in and sign out.
  • Regular Users authorized to CRUD their appointments.
  • Admin authorized to CRUD Regular Users appointments.
  • Admin authorized to CRUD Doctor Inofrmation.
  • First name and Last name is authenticated to be in the Arabic language.
  • Phone is authenticated to be in 10 characters
  • An appointment authenticated to date and time.
  • The website connected to the clinic domain
  • Applity to Search Doctors
Development Process:

The website was built on Ruby on Rails. The Project were divided to Front-End and Back-End through team members. Source controlling was used to merge all work. A database was built using Mongoid. Styling was done using Bootsrtap library.
