Tic Tac Toe!

This website created for children to play the Tic Tac Toe game.

Tic Tac Toe Website Overview:

This game was created as a part of the learning process. The main goals of the project is to be able to build a Tic Tac Toe game with 3X3 grid (board) from scratch.

How to play:

Tic Tac Toe or Xs and Os is a game for two players on a 3X3 game board, which means 9 cells. For the first round, player1 is X, and player2 is O. Then for the second round, it will switch. The player will take turns marking the cells. To win a Tic Tac Toe game you need to get three tokens in a horizontal, vertical, or diagonal row.

Technologies :

The web application built from scratch by useing HTML , CSS, Javascript and JQuery.

  • HTML: Was used for the table of the board game.
  • CSS: Was used for the main styling.
  • Javascript: Was used for the functionality of the game, using DOM manipulation.
  • JQuery: Was used for the functionality of the game, using DOM manipulation.
  • Source Control: was used for interaction, management and upload changes on code to Git repository
  • Command Line: used for interacting with the computer, navigating the filesystem.
  • Adobe XD: Was used for Wireframe.
  • GitHub: Was used for deployment.
  • SweetAlert: used for modals popups.
  • Animate.css: used for animations.
  • Visual Studio: Was used for coding.
  • Google Chrome Web Browser: Was used to view the website.
  • Google Chrome Developer Tools: Was used to debug and solve problems in the code.
Features :
  • User could keep track of the score in a multiple game rounds with a win and lose counter.
  • User can click on a square to add X first and then O, and so on.
  • User will see a message when it's a win, lose or tie after each game.
  • User can't click the same square twice.
  • User can't continue playing once it's a win, lose, or tie.
  • The game will switch turns between X and O.
  • Applity to start the game without refreshing the page.
  • Applity to restart the game while in a middle of the game.
  • The website deployet into GitHub.
