EraCure

The website is a Landing page for an EraCure app that provides Home Physical therapy services.

EraCure Website Overview:

This website was created as a part of the learning process. The main goals of the project is to be able to build website from scratch with using of HTML, CSS, Javascript and JQuery. The website contains information about EraCure and the services they provide.

Technologies :

The web application built from scratch by useing HTML, CSS, Javascript and JQuery.

  • HTML: Was used for the content.
  • CSS: Was used for the main styling.
  • Javascript: Was used for the functionality, using DOM manipulation.
  • JQuery: Was used for the functionality, using DOM manipulation.
  • Source Control: Was used for interaction, management and upload changes on code to Git repository
  • Command Line: Was used for interacting with the computer, navigating the filesystem.
  • GitHub: Was used for deployment.
  • Visual Studio: Was used for coding.
  • Google Chrome Web Browser: Was used to view the website.
  • Google Chrome Developer Tools: Was used to debug and solve problems in the code.
  • Adobe XD: Was used for Wireframe.
Features :
  • Website built in one page.
  • Make progress bar shows up and move when scrolling down or up .
  • Toggle information in about us page
  • The ability to change the name in contact us page after submiting
  • Fade in pictures in whyus page when reaching the sections
  • The website deployet into GitHub.
Future plans :
  • Improve the responsive design.
  • The ability to send and receive messages through a server.
